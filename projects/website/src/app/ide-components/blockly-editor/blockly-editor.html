How C-SLIDE blocks work

  1. Movement blocks control your simulator truck path.
  2. R3 Kit blocks generate LED, motor, sensor, and servo sketch code.
  3. Lights blocks manage colorful virtual and physical light effects.
  4. Text and Sound blocks add serial messages and tone feedback.
  5. Use top toolbar Save/Open to keep projects in your account.
